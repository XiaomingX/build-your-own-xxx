# AI + 量化交易策略 (Quant Trading Strategies)

这里收集了 AI 在量化交易、算法交易、金融强化学习和市场预测领域的开源替代方案和工具。

## 综合量化平台与框架 (Comprehensive Platforms & Frameworks)

*   **[Qlib](https://github.com/microsoft/qlib)**
    *   **描述**: Microsoft 推出的面向 AI 的量化投资平台。
    *   **用途**: 提供全流程的量化研究支持，包括数据处理、模型训练（支持各类 ML/DL 模型）和回测。

*   **[Nautilus Trader](https://github.com/nautilus-trader/nautilus_trader)**
    *   **描述**: 基于 Rust 和 Python 构建的高性能算法交易平台。
    *   **用途**: 事件驱动的交易策略开发，适合高频和低延迟需求，易于集成 AI 模型。

*   **[Lean](https://github.com/QuantConnect/Lean)**
    *   **描述**: QuantConnect 的开源算法交易引擎。
    *   **用途**: 支持 C#, Python，拥有庞大的社区和数据支持，适合云端和本地回测。

## 强化学习交易 (Reinforcement Learning for Trading)

*   **[FinRL](https://github.com/AI4Finance-Foundation/FinRL)**
    *   **描述**: 专门用于金融交易的深度强化学习库。
    *   **用途**: 提供针对股票、加密货币等市场的 DRL 算法和环境，大大降低了将 RL 应用于交易的门槛。

*   **[TensorTrade](https://github.com/tensortrade-org/tensortrade)**
    *   **描述**: 用于构建、训练和部署交易算法的强化学习框架。
    *   **用途**: 模块化设计，允许自定义行动方案和奖励机制，基于 TensorFlow 和 OpenAI Gym。

## 加密货币交易机器人 (Crypto Trading Bots)

*   **[Freqtrade](https://github.com/freqtrade/freqtrade)**
    *   **描述**: 领先的加密货币开源交易机器人，支持 Python。
    *   **用途**: 包含 FreqAI 模块，利用机器学习进行自适应预测，支持策略回测和实盘交易。

*   **[Hummingbot](https://github.com/hummingbot/hummingbot)**
    *   **描述**: 专注于做市 (Market Making) 和套利策略的开源客户端。
    *   **用途**: 为中心化和去中心化交易所提供流动性，支持编写自定义策略脚本。

*   **[Jesse](https://github.com/jesse-ai/jesse)**
    *   **描述**: 一个用于加密货币交易策略的研究和回测框架。
    *   **用途**: 简单易用的 API，支持这导入蜡烛图数据进行精确回测。
