Langfuse.com 是一个开源的 LLMOps 平台，主要用于 LLM 应用的可观测性和工程化管理。 [langfuse](https://langfuse.com/cn)

## 主要功能
它提供追踪（trace）、监控成本/延迟/质量、提示管理、评估（包括 LLM-as-judge 和人工标注）、测试（A/B 实验）和用户行为分析，支持 LangChain、LlamaIndex、OpenAI 等框架集成，可自托管或云端部署。 [wiki.hiwepy](https://wiki.hiwepy.com/docs/langfuse)

## 开源替代品
- **Phoenix（Arize Phoenix）**：开源 LLM 追踪和评估工具，强调生产环境调试，支持自托管。 [braintrust](https://www.braintrust.dev/articles/langfuse-alternatives-2026)
- **TensorZero**：完全开源替代，包含 LLM 网关、可观测性、优化和实验功能，可与 Langfuse 互补使用。 [tensorzero](https://www.tensorzero.com/docs/comparison/langfuse)
- **Helicone**：开源代理工具，专注成本追踪和多提供商支持，但评估功能较弱。 [braintrust](https://www.braintrust.dev/articles/langfuse-alternatives-2026)
Langfuse 本身已是 LangSmith 的流行开源替代，且自托管免费，企业版有额外功能。 [opensource](https://opensource.zone/alternatives/langsmith)
