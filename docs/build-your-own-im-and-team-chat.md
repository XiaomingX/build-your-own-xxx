# 开源即时通讯工具 (Open Source IM Alternatives)

这里收集了类似微信、Telegram、Discord 的开源即时通讯和团队协作工具，支持自托管，保障数据隐私。

## 综合即时通讯 (Global/Private Messenger) - 微信/Telegram 替代

*   **[Signal](https://github.com/signalapp)**
    *   **描述**: 以隐私为核心的加密通讯应用，行业标准。
    *   **用途**: 个人隐私通讯，代替 WhatsApp/微信/Telegram 的私聊功能。

*   **[Element (Matrix)](https://github.com/element-hq/element-web)**
    *   **描述**: 基于 Matrix 协议的旗舰客户端，支持端到端加密和去中心化。
    *   **用途**: 可替代 Telegram，支持群聊、VoIP 和跨平台同步，适合极客和隐私用户。

*   **[Session](https://github.com/oxen-io/session-desktop)**
    *   **描述**: 基于 Signal 协议的分叉版本，支持洋葱路由，无需手机号注册。
    *   **用途**: 极致隐私需求的匿名通讯。

*   **[Berty](https://github.com/berty/berty)**
    *   **描述**: P2P 离线优先的通讯工具，基于 IPFS。
    *   **用途**: 在无网络或受限网络环境下的蓝牙/本地通讯。

## 团队协作与社区 (Team & Community) - Discord/Slack 替代

*   **[Rocket.Chat](https://github.com/RocketChat/Rocket.Chat)**
    *   **描述**: 功能最全面的开源协作平台，支持全渠道客服集成。
    *   **用途**: 企业内部沟通、Discord 替代，支持广泛的插件和自定义。

*   **[Mattermost](https://github.com/mattermost/mattermost)**
    *   **描述**: 高安全性的 Slack 开源替代品，Go 语言编写，性能优异。
    *   **用途**: 开发者团队协作，DevOps 集成。

*   **[Zulip](https://github.com/zulip/zulip)**
    *   **描述**: 独特的“话题流”模式，结合了即时通讯和电子邮件的组织性。
    *   **用途**: 适合需要处理大量信息流的分布式团队。

*   **[Spacebar (原 Fosscord)](https://github.com/spacebarchat/spacebar)**
    *   **描述**: 旨在完全兼容 Discord API 的开源后端。
    *   **用途**: 自托管的 Discord 体验，社区建设。

*   **[Revolt](https://github.com/revoltchat)**
    *   **描述**: 轻量级的 Discord 替代品，界面简洁。
    *   **用途**: 专注于游戏玩家和社区的简约聊天平台。

## 其他特色工具

*   **[Nextcloud Talk](https://github.com/nextcloud/spreed)**: 集成在 Nextcloud 中的音视频会议和聊天。
*   **[Jitsi Meet](https://github.com/jitsi/jitsi-meet)**: 纯开源的视频会议解决方案。
