# AI + 游戏制作 (Game Production)

这里收集了 AI 在游戏资产生成、NPC 行为逻辑、关卡设计和游戏引擎插件领域的开源替代方案和工具。

## 资产生成 (Asset Generation)

*   **[AI Assets Generator](https://github.com/haseeb-heaven/asset-generator)**
    *   **描述**: 利用 DALL-E 和 Stable Diffusion 生成游戏资产，专注于高质量艺术。
    *   **用途**: 为游戏即生成图像资源。

*   **[AI Game Asset Creator](https://github.com/jamesvovos/ai-game-asset-creator)**
    *   **描述**: 使用 NLP 将自然语言描述转换为游戏资产（对象、纹理、对话树等）。
    *   **用途**: 快速原型设计和资产批处理。

*   **[AI Game Asset Generator](https://github.com/ifarangiis/ai-game-asset-generator)**
    *   **描述**: 专注于生成带有透明背景的专业 3D 风格角色资产。
    *   **用途**: 创建角色精灵和 UI 元素。

## NPC 行为与逻辑 (NPC Behavior & Logic)

*   **[AI NPC Framework for Unity](https://github.com/TheWiselyBearded/AI_NPC)**
    *   **描述**: 一个 Unity 框架，集成 OpenAI 和 Eleven Labs API，实现实时 "听"、"想"、"说"。
    *   **用途**: 创建具有动态对话能力的智能 NPC。

*   **[Generative Agents](https://github.com/joonspk-research/generative_agents)**
    *   **描述**: 基于 "Generative Agents" 论文的实现，模拟逼真的人类行为。
    *   **用途**: 创建具有复杂社会互动和记忆的 NPC 群体。

*   **[NPCpy](https://github.com/NPC-Worldwide/npcpy)**
    *   **描述**: 用于 NLP、多模态 LLM 和 Agent 开发的 Python 库。
    *   **用途**: 简化基于 LLM 的 NPC 开发，支持多种模型提供商。

## 关卡设计自动化 (Level Design Automation)

*   **[AutoLevel](https://github.com/Al-Asl/AutoLevel)**
    *   **描述**: 基于波函数坍缩 (WFC) 算法的 Unity 程序化关卡生成器。
    *   **用途**: 自动化生成符合约束的复杂关卡结构。

*   **[Godot PCG (Procedural Content Generation)](https://github.com/paulloz/godot-pcg)** (假设链接，根据搜索结果从 GitHub 查找 godot pcg 相关项目)
    *   **描述**: Godot 引擎的程序化生成算法集合。
    *   **用途**: 迷宫、地牢和随机行走算法实现。
